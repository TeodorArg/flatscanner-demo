# GitHub repository bootstrap checklist

- определена default branch
- включены branch rules / protection для основной ветки
- merge в основную ветку идет через PR
- required checks совпадают с реальными workflow/job names
- GitHub Actions permissions определены явно
- repository variables созданы и задокументированы
- repository secrets созданы и задокументированы
- runner policy определена
- merge policy определена
- правила в GitHub согласованы с `AGENTS.md` и docs
